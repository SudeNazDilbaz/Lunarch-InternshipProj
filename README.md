# Lunarch Secure Uploader

Lunarch Secure Uploader is a Python-based security module designed to improve file upload safety in web applications.

## Project Status
Planning phase

## Main Focus
- Secure file validation
- Filename sanitization
- Path traversal prevention
- Safe file storage

## Project Structure

The project is organized into separate modules to keep each security task independent and reusable.

- validator.py
- sanitizer.py
- permissions.py
- uploader.py
